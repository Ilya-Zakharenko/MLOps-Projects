# <center> **MLOps Projects**

This folder contains projects focused on **Machine Learning Operations (MLOps)** — the practice of deploying, monitoring, maintaining, and scaling machine learning models in production environments.

Here I demonstrate skills in model serving, monitoring, metrics collection, visualization, and production-ready ML pipelines.

---

### **Projects**

- **[Wine Quality Model Monitoring](Wine_Quality_Model_Monitoring/)**  
  End-to-end MLOps project: training a wine quality classification model with `GridSearchCV`, exposing model metrics via Prometheus, and building a comprehensive monitoring dashboard in Grafana.



---

### **Technologies Commonly Used**

- **Model Training**: scikit-learn, pandas, numpy
- **Model Serving**: Flask / FastAPI
- **Monitoring**: Prometheus, Prometheus Python Client
- **Visualization**: Grafana
- **Others**: Docker, CI/CD basics, joblib
